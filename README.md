# SEG 3125 - Design and Analysis Of User Interfaces
# Assignment 2 - UCD for Development of a Service Business Site

Welcome to my second assignment! This assignment contains a high fidelity prorotype named
QuickCutz, a local affordable hair dresser that I created that is theoretically located near my
hometown. I included a variety of elements learned in class up until this point, so I invite
you to browse the site and enjoy!

By carrying out this assignment, I:  
• Better understood prototyping approaches, in particular the development of mockups with
the help of tools, such as Figma.  
• Practiced the iterative design approach  
• Continued exploring visual design as started in the last assignment  
• Became familiar with HTML, CSS, Bootstrap and JavaScript for building the business web site.

The high-fidelity prototype meets the following technical requirements:  
a. Is developed using a JavaScript framework (e.g., React)  
b. Is interactive (responds to commands – buttons, list selections)  
c. Is accessible through my portfolio (from assignment 1)

My high-fidelity prototype is guided by the following visual design principles:  
a. The use of a color theme to insure consistency  
b. Typography  
c. Iconography  
d. The layout and the use of negative space  
e. The principles of contrast, scale, balance, visual hierarchy.
